# GameNewsHub - 個人用ニュースアグリゲーター (Cloud デプロイ手順)

このプロジェクトは、ゲームメディアを含む様々なWebサイトからニュース記事（RSS/スクレイピング）を非同期に取得し、重複を自動統合して表示する個人用のWebアプリケーションです。

静的ファイル（HTML, CSS, JS）のみで構築されているため、以下のような無料のホスティングサービスを利用して簡単にCloud上にデプロイし、スマートフォンから閲覧することができます。

---

## ローカルでの実行方法（PC上で試す場合）
CORS制限や相手サイト（AUTOMATON等）のボット防止フィルターによる取得エラーを回避するため、PC上で専用の「ローカルプロキシサーバー」を起動して接続することができます。

1. **パッケージのインストール**:
   プロジェクトフォルダ内で以下を実行し、必要なパッケージ（Express, Axiosなど）をインストールします。
   ```bash
   npm install
   ```
2. **サーバーとアプリの起動**:
   以下のコマンドを実行すると、中継サーバー（ポート5000）とクライアント（Vite）が同時に起動します。
   ```bash
   npm run dev
   ```
   起動後、ブラウザで `http://localhost:5173` （またはViteが起動したアドレス）にアクセスします。
   ローカルで動作している間は、`app.js` が自動検知してローカルサーバーを経由し、一般のブラウザと同じヘッダーを偽装して各メディアから直接ニュースを取得するため、エラーを完全に回避できます。

## デプロイ方法

### 方法 1: Netlify (最も簡単、GitHub不要)
フォルダをブラウザにドラッグ＆ドロップするだけで、数秒で公開できます。

1. **[Netlify](https://www.netlify.com/)** にログイン（または無料アカウント作成）します。
2. ログイン後、ダッシュボードの **"Sites"** ページに移動します。
3. ページ下部にある **"Deploy manually"** の枠内に、このプロジェクトフォルダ（`game-news-tracker`）を丸ごとドラッグ＆ドロップします。
4. 数秒でアップロードが完了し、`https://xxxxxx.netlify.app` のような公開用URLが生成されます。
5. スマートフォンからそのURLにアクセスすれば、いつでもニュースを確認できます！

---

### 方法 2: Vercel (高速かつ簡単)
GitHubと連携して、コードを更新するたびに自動で再デプロイされる構成です。

1. このフォルダをご自身の **GitHub** リポジトリにプッシュします。
2. **[Vercel](https://vercel.com/)** にログインし、**"Add New" -> "Project"** を選択します。
3. 作成したGitHubリポジトリを選択（Import）します。
4. 設定はデフォルトのままで **"Deploy"** ボタンをクリックします。
5. デプロイが完了すると、自動的に独自サブドメイン（`https://xxxx.vercel.app`）が割り当てられます。

---

### 方法 3: GitHub Pages (完全無料・GitHub標準機能)
GitHubリポジトリの設定画面から直接有効化する方法です。

1. コードをご自身の **GitHub** リポジトリにプッシュします。
2. リポジトリの **"Settings"** タブを開きます。
3. 左メニューの **"Pages"** を選択します。
4. **"Build and deployment"** の **"Branch"** で `main` (または `master`) ブランチの `/ (root)` を選択し、**"Save"** をクリックします。
5. 数分後、`https://<ユーザー名>.github.io/<リポジトリ名>/` にてWebサイトが公開されます。

## スマートフォンでの快適な使い方

1. 公開されたURLにスマートフォンのブラウザ（Safari、Chromeなど）でアクセスします。
2. ブラウザのメニューから **「ホーム画面に追加」** を選択します。
3. ホーム画面にアプリアイコンのように登録され、ネイティブアプリのようなフルスクリーン感覚でニュースアグリゲーターを起動できるようになります。

## 主な機能と特徴
- **自動重複排除**: Dice's Coefficient（類似度アルゴリズム）を使用し、異なるメディアが報じる同一のニュース記事を1つのカードに自動統合し、それぞれのメディアへのリンクを並べて表示します。
- **期間フィルター**: デフォルトは「8時間以内」に設定されており、12時間、24時間、3日間から選択可能です。
- **非同期・並行取得**: 各ソースから並行して記事を非同期取得し、ロードが完了したメディアから順次画面をリアルタイムに更新・表示します。
- **カテゴリ管理機能**: ⚙（管理）ボタンから、新しいカテゴリを追加し、独自のRSSフィードURLを登録できます。初期状態で搭載されている「ゲーム」カテゴリには以下の主要5サイトが自動登録されています。
  - ファミ通.com
  - 電ファミニコゲーマー
  - 電撃オンライン
  - 4Gamers
  - AUTOMATON
